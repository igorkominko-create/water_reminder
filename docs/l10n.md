# Localization (en / uk)

Uses Flutter **gen-l10n** with ARB files and `intl` for dates.

## Structure

```
l10n.yaml                 # generator config
lib/l10n/
  app_en.arb              # English (template)
  app_uk.arb              # Ukrainian
  app_localizations.dart  # generated — do not edit
```

## pubspec.yaml

```yaml
dependencies:
  flutter_localizations:
    sdk: flutter
  intl: ^0.20.2

flutter:
  generate: true
```

## Regenerate after editing ARB

```bash
flutter gen-l10n
# or
flutter pub get
```

## Usage in UI

```dart
import 'package:water_reminder/l10n/app_localizations.dart';

final l10n = AppLocalizations.of(context);
Text(l10n.homeTitle);
Text(l10n.mlToGo(500));
```

## App wiring

- `main.dart` — `AppLocale.configure()` sets `Intl` from device language.
- `WaterApp` — `localizationsDelegates`, `supportedLocales`, `localeResolutionCallback`.
- Language follows **system locale** (`uk` → Ukrainian, otherwise English).
