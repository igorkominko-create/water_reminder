# Water Reminder

Minimal Flutter hydration tracker with a focus on **iOS Lock Screen / Home Screen widgets** and **Android home screen widgets**.

## Run locally

```bash
cd water_reminder
flutter pub get
flutter run
```

## What's included (v0.1)

- Daily water goal and quick-add buttons (150–500 ml)
- Circular progress UI (minimal blue theme, DM Sans)
- Local persistence (`shared_preferences`)
- `home_widget` bridge — data sync ready for native widgets
